== Description ==

Fliiby is a comprehensive video, image, music, document and other file types hosting platform that offers everything your business needs for online video publishing, photography publishing, music publishing, document publishing. With Fliiby, publishers have an all-in-one solution to encode, host, syndicate, stream, and monetize their digital content. It's free and you earn revenue for views of content.




The Fliiby WordPress HTML5 Player plugin is an easy way for you to integrate with your Fliiby account and manage your files such as videos, images, music, documents and other file types,  and post them without leaving your WordPress admin CMS.


The Fliiby platform handles SD and HD FLV & H.264 encoded video formats like MP4, M4V, M4A, MOV, MP4V etc... HTML 5 video support enables cross browser and cross platform (desktop, tablet, mobile) playback, on top of that our players support almoust all major file extensions, if you can't find one that you want to upload please submit ticket.

== Installation ==

1. Use the WordPress plugin installer to install the plugin.  Alternatively, you can just extract the folder in our download package and upload it to your plugin directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Read instruction and set default's after clicking the new Fliiby menu item that shows up in your admin panel. 
1. To embed files simply paste Fliiby link into your post. Make sure each link is on its own line. Or use [fliiby]url[/fliiby] markup
1. Use Fliiby button (next to media library button) in posts to upload, search and select files for embed.
